## Why this matters

The presentation IS the hiring signal. It demonstrates: technical depth, decision-making under constraint, communication, teamwork, and the ability to defend a claim under questioning. Nothing else in the program does all five.

## Time budget for today

| Block | Activity |
|---|---|
| Morning | Final dry run (each team, 30 min slot) |
| Lunch | - |
| Afternoon | Presentations: 25 min/team (15 present + 10 Q&A) + 5 min reset between |
| End | Panel deliberation; same-day verdict for each team |

- [ ] Final dry run completed for every team

- [ ] Peer-feedback forms completed for all teams

## Presentation discipline

1. **Time-keep ruthlessly.** 15 minutes means 15 minutes. The panel signals at 14 and cuts at 15.
2. **Every team member presents at least one slide section.** Roles assigned Day 47.
3. **The claim sentence is uttered verbatim** on slide 10. Don't paraphrase. Don't soften.
4. **Q&A is part of the grade.** Three rules:
   - If asked a question you didn't prepare for: **think before answering.** A 5-second pause is fine.
   - If you don't know: "We didn't test that" beats making something up. Followed by what you *would* do to find out.
   - If a teammate has the answer, hand off cleanly: "Anita has the cost numbers."

## What the rubric scores

Per [Day-49 Assessment Rubric](../../../../planning/source-material/Capstone/Day-49-Assessment-Rubric.md), each of:

| Dimension | What earns top marks |
|---|---|
| Use case clarity | Specific user, specific task, real value |
| Technical depth | Phase 1 vocabulary throughout; correct reasoning under questioning |
| Methodology rigor | Sweep over relevant axes, controls for confounds, eval plan executed |
| Evidence quality | Comparison table is complete, costs calculated correctly |
| Recommendation strength | Claim sentence is defensible; tradeoff acknowledged honestly |
| Q&A handling | Composed, accurate, hand-offs work, admits limits |
| Teamwork | Visible contribution from each member |
| Presentation polish | Slides readable, time respected, narrative arc clear |

8 dimensions. Each scored. Final grade is the rubric total; see template.

## Peer assessment

After each presentation, every other student fills a short peer-feedback form. **This counts** (small but real weight). Be specific, be kind, be honest.

## Common failure modes (don't)

| Failure | Fix |
|---|---|
| Reading the slides | Look at the panel; speak to them |
| Over-running | Cut content in dry run, not on stage |
| One person does 12/12 slides | Already assigned Day 47; stick to it |
| Defensive Q&A | Curiosity > defense. "Great question, here's what we found..." |
| Pretending you tested something you didn't | Panel will catch it. Be honest. |
| Burying the claim | Slide 10 is large + simple + memorable |
